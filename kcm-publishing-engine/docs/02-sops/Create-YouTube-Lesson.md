# Create YouTube Lesson SOP

## Purpose

This SOP explains how to create a complete KCM long-form YouTube math lesson from idea to published video.

## Goal

Create beginner-friendly, visually clear math lessons that help students understand the topic and feel more confident.

## Standard Lesson Length

15–30 minutes

## Production Pipeline

Idea → Research → Outline → Script → PowerPoint → Practice Problems → Voiceover → Export → QA → Publish

## Step 1: Choose the Topic

Pick one focused math topic.

Examples:

- Factoring
- Solving equations
- Graphing lines
- Quadratics
- Fractions
- Percentages
- Exponents

The topic should be specific enough to teach clearly in one video.

Good:

- Factoring Trinomials
- Solving Two-Step Equations
- Graphing Lines in Slope-Intercept Form

Too broad:

- Algebra
- Geometry
- Everything About Math

## Step 2: Define the Student

Before outlining, define who the lesson is for.

Questions:

- What grade level is this?
- What does the student already know?
- What usually confuses them?
- What should they be able to do by the end?

Example:

This lesson is for Algebra I students who know basic multiplication but struggle with factoring trinomials.

## Step 3: Create the Lesson Outline

Use this structure:

1. Hook
2. Why the topic matters
3. Key vocabulary
4. Main concept
5. Example 1: Easy
6. Example 2: Medium
7. Example 3: Challenge
8. Common mistakes
9. Practice problems
10. Recap
11. Call to action

## Step 4: Write the Script

The script should sound warm, clear, and encouraging.

Tone:

- Friendly
- Calm
- Beginner-friendly
- Confident
- Encouraging

Avoid:

- Overly academic language
- Rushing
- Too many concepts at once
- Assuming the student already understands

## Step 5: Build the PowerPoint

Each slide should teach only one idea.

Recommended slide types:

- Title slide
- Topic overview
- Vocabulary slide
- Formula or rule slide
- Step-by-step example slides
- Practice problem slides
- Common mistake slides
- Recap slide
- CTA slide

## Step 6: Add Practice Problems

Include:

- 3 guided examples
- 3 independent practice problems
- 1 challenge problem

Each problem should include a solution slide.

## Step 7: Quality Assurance

Before recording, check:

- All math is correct
- Steps are clear
- Text is readable
- Slides are not overcrowded
- Examples increase in difficulty
- The lesson has a clear beginning, middle, and end

## Step 8: Record Voiceover

Voiceover should be:

- Slow enough for beginners
- Warm and encouraging
- Clear and confident
- Natural, not robotic

Pause after important steps.

## Step 9: Export

Export final files to:

```text
exports/video
exports/pdf
exports/pptx
```

Recommended naming:

```text
YYYY-MM-DD_kcm_youtube_topic_v1
```

Example:

```text
2026-07-04_kcm_youtube_factoring-basics_v1
```

## Step 10: Publish

Before publishing, prepare:

- Title
- Description
- Thumbnail
- Tags
- Pinned comment
- Related product link
- Newsletter link
- Tutoring/app link

## Step 11: Repurpose

Each long-form lesson should become:

- 3–5 Shorts
- 1 worksheet
- 1 email newsletter
- 1 Threads post
- 1 Pinterest pin
- 1 future product asset

## Success Criteria

A successful KCM lesson is:

- Accurate
- Clear
- Beginner-friendly
- Visually clean
- Encouraging
- Repurposable
- Connected to a product or service funnel

## Motto

Teach clearly. Encourage deeply. Build confidence.
